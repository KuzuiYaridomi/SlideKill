**What is Slide KiLL?**

Slide KiLL is an in-development parkour action game :D
The physics are kind of wanky, chaotic, and high-speed, because its my first time dealing with so many features at once and may end up creating unpredictable moments.

You can slide at insane speeds, bounce off surfaces, and even ram into enemies so hard that they instantly die.
Your movement, momentum, and timing create the main combat system.
